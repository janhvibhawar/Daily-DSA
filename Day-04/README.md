# Armstrong Number Check

## Problem
Check whether a given number is an **Armstrong number**.

An Armstrong number is a number that is equal to the sum of its digits raised to the power of the total number of digits.

## Approach
- Count the number of digits.
- Extract each digit and raise it to the power of digit count.
- Sum all values and compare with the original number.

## Time Complexity
O(d), where d is the number of digits.

## Space Complexity
O(1)
